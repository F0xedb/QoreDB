Interface IPager

Namespace
QoreDB.StorageEngine.Pager.Interfaces
Assembly
QoreDB.dll

Defines the core contract for a pager component, which is responsible for reading and writing fixed-size pages to and from an underlying data stream. This interface abstracts the direct interaction with the data source.

GetPage(int)

Retrieves a page by its ID, utilizing an in-memory cache for performance. If the page is present in the cache (a cache hit), it is returned directly. If not (a cache miss), it is read from the underlying stream, loaded into the cache, and then returned.
