Class LeafNode<TKey, TValue>

Namespace
QoreDB.StorageEngine.Index.Nodes

Represents a leaf node in the B+ Tree. Leaf nodes store the actual key-value pairs (entries). All leaf nodes are linked together in a doubly-linked list for efficient range scans.

public class LeafNode<TKey, TValue> : Node<TKey>, INode<TKey> where TKey : IComparable<TKey>
