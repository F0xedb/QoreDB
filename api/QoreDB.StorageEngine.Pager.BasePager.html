Class BasePager

Namespace
QoreDB.StorageEngine.Pager
Assembly
QoreDB.dll

Provides a base implementation for a pager that manages reading and writing pages to an underlying stream, with an integrated LRU (Least Recently Used) caching mechanism. This class is abstract and must be inherited by a concrete pager implementation that provides the specific stream (e.g., FileStream or MemoryStream).
